# customercli
Customer Management System
1. clone
2. npm install
3. run node command.js
4. add customer
     - node command.js add
5. find customer
     - node command.js find <firstname>
6. list customer
     - node command.js list
7. update customer
     - node command.js update <_id>
8. remove customer
     - node command.js remove <_id>
